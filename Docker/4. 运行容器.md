# 4. 运行容器

运行一个容器：

```bash
docker run nginx
```

常用参数：

|参数|作用|
|---|---|
|`-d`|后台运行|
|`-p`|映射端口|
|`--name`|给容器起名字|
|`-v`|挂载数据卷|

例如，运行 Nginx 并映射端口：

```bash
docker run -d -p 8080:80 --name mynginx nginx
```

进入容器：

```bash
docker exec -it mynginx bash
```

停止容器：

```bash
docker stop mynginx
```

删除容器：

```bash
docker rm mynginx
```
